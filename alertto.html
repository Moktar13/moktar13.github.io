alertTO

An iOS application aimed to inform the residents of the Greater Toronto Area about recent incidents near them.

alertTO is an interactive and informative application designed to keep residents of the Greater Toronto Area (GTA) informed about recent incidents. The application interfaces with the Toronto Police Service, ensuring that the information provided is accurate and up-to-date.

This app, published to the App Store, utilizes a variety of technologies and services including Firestore for data storage, Cloud Functions for backend logic, AdMob for ad integration, Create ML for machine learning capabilities, and asynchronous API calls for fetching data. It also incorporates StoreKit2 for in-app purchases, SwiftUI for its user interface, MVVM architecture for code organization, Push Notifications for user engagement, UserDefaults for local data storage, Location Services for location-based features, and Notification Services for delivering notifications to users.
